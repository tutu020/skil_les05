# Werken met Git en GitHub

![Git Workflow](https://miro.mediate.com/v2/resize:fit:1400/1*T5-ul-2hDJ97Ho5q4t6Fig.png)

## Onderdelen

### Starten met locale repository
- Maak een nieuwe map op je computer
- Open terminal/command prompt in die map
- Initialiseer Git repository

### Werken met online repository
- Maak account op GitHub.com
- Maak nieuwe repository aan
- Koppel locale repository aan online repository

## Acties die je uit moet voeren

- Git installeren op je computer
- GitHub account aanmaken
- Repository initialiseren
- Bestanden toevoegen en committen
- Repository pushen naar GitHub

### Extra informatie

1. Installeer Git van git-scm.com
2. Open terminal/command prompt
3. Navigeer naar je projectmap: `cd pad/naar/je/map`
4. Initialiseer Git: `git init`
5. Voeg bestanden toe: `git add .`
6. Maak eerste commit: `git commit -m "Eerste commit"`
7. Koppel aan GitHub: `git remote add origin [jouw-github-link]`
8. Push naar GitHub: `git push -u origin main`
